## Cross-Project Learning

When a retro surfaces a lesson about the **template itself** (not just this project):
1. Note it in "Template Feedback" above
2. After sprint wraps, update `~/clawd/drafts/workbench-template-v2/` with the improvement
3. This way, future projects benefit from every project's learnings
